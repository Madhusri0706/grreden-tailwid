# Greenden 🌱

Greenden is a simple responsive plant website built using HTML and Tailwind CSS.

## Features

- Responsive Navbar
- Hero Section
- About Section
- Best Seller Products
- Customer Reviews
- Newsletter Section
- Product Page
- Contact Page
- Hover Effects

## Technologies Used

- HTML5
- Tailwind CSS

## Pages

- Home Page (`index.html`)
- Products Page (`product.html`)
- Contact Page (`contact.html`)
